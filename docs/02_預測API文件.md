---

## 文件說明

本 API 提供系統兩種功能：
1. **預測文本內容之分類順序**
2. **jieba 斷詞權重結果**

兩者皆依據給定的文本資料和要求輸出之筆數，按照權重比例或預測信心值輸出結果。

---

## 服務通用規格

- 傳輸資料皆採用 **JSON 格式**，**UTF-8** 編碼
- 規格一律使用 `application/json`
- Port 指定為 **80**

---

## API 1：predict — 預測文本分類

| 項目   | 說明                   |
| ---- | -------------------- |
| 代碼   | `new_textCNN/run.py` |
| 說明   | 預測文本分類               |
| 服務路徑 | `位址:80/predict`      |

### REQUEST

- **提交方式**：POST

| 提交參數        | 說明   | 型態     |
| ----------- | ---- | ------ |
| `preString` | 預測文本 | String |
| `preNum`    | 預測數量 | String |

**提交範例：**

```json
{
  "preString": "檢舉補習班違規,渠反映太麻里鄉太麻里街上「賢儒補習班」(民眾不知正確地址)，裡面設有安親班、音樂班及補習班，其使用坪數未達標準，且老師(表示是主任)有體罰學生的行為，建請查處。(個資極需保密，不得提供給該補習班相關人員)(已提供代碼)",
  "preNum": 3
}
```

### RESPONSE

| 回傳參數      | 說明           | 型態    |
|---------------|----------------|---------|
| `id`          | 預測結果之 id  | Integer |
| `ou`          | 預測結果       | String  |
| `probability` | 預測信心值     | String  |

**回傳範例：**

```json
{
  "case": [
    {
      "id": 126,
      "ou": "警察局",
      "probability": "0.5066519975662231"
    },
    {
      "id": 26,
      "ou": "社會處",
      "probability": "0.2029060423374176"
    },
    {
      "id": 96,
      "ou": "教育處",
      "probability": "0.179448202252388"
    }
  ]
}
```

## API 2：TFIDF — jieba 斷詞結果

| 項目     | 說明                 |
|----------|----------------------|
| 代碼     | `new_textCNN/run.py` |
| 說明     | Jieba 斷詞結果       |
| 服務路徑 | `位址:80/TFIDF`      |

### REQUEST

- **提交方式**：POST

| 提交參數        | 說明     | 型態     |
| ----------- | ------ | ------ |
| `preString` | 文本     | String |
| `preNum`    | 斷詞輸出數量 | String |

**提交範例：**

```json
{
  "preString": "檢舉補習班違規,渠反映太麻里鄉太麻里街上「賢儒補習班」(民眾不知正確地址)，裡面設有安親班、音樂班及補習班，其使用坪數未達標準，且老師(表示是主任)有體罰學生的行為，建請查處。(個資極需保密，不得提供給該補習班相關人員)(已提供代碼)",
  "preNum": 3
}
```

### RESPONSE

| 回傳參數      | 說明     | 型態   |
|---------------|----------|--------|
| `ou`          | 斷詞結果 | String |
| `probability` | 斷詞權重 | String |

**回傳範例：**

```json
{
  "case": [
    {
      "ou": "補習",
      "probability": "1.2261300002974358"
    },
    {
      "ou": "太麻里",
      "probability": "0.6130650001487179"
    },
    {
      "ou": "檢舉",
      "probability": "0.30653250007435895"
    }
  ]
}
```
